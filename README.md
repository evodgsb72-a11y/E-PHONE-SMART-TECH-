# 📱 E-PHONE SMART TECH – Phone Shop Management App

## 🚀 Description
E-PHONE SMART TECH is an Android application built using Java and SQLite for managing a phone shop. It helps track phones, sales, and inventory easily.

## 🧱 Tech Stack
- Java (Android)
- XML (UI Design)
- SQLite (Local Database)
- AndroidIDE / Android Studio

## 📌 Features (Phase 1)
- Login System
- Dashboard
- Add Phone
- View Phones
- SQLite Database
- Edit/Delete (coming soon)
